# Sprunki Character Customizer

Welcome to the Sprunki Character Customizer! This mobile application allows users to create and customize their own Sprunki characters. With a variety of options for outfits, accessories, colors, and expressions, users can unleash their creativity and share their unique characters with others.

## Features

- **Character Selection**: Choose from a roster of Sprunki characters.
- **Customization Interface**: Modify character outfits, accessories, colors, and facial expressions.
- **Gallery**: View a gallery of saved characters and their customizations.
- **User Profile**: Manage your profile and saved characters.
- **Sharing Options**: Share your customized characters with friends.

## Getting Started

To get started with the Sprunki Character Customizer, follow these steps:

1. **Clone the Repository**:
   ```
   git clone https://github.com/yourusername/sprunki-character-customizer.git
   ```

2. **Navigate to the Project Directory**:
   ```
   cd sprunki-character-customizer
   ```

3. **Install Dependencies**:
   ```
   npm install
   ```

4. **Run the Application**:
   ```
   npm start
   ```

   This will start the Expo development server. You can then scan the QR code with the Expo Go app on your mobile device to view the application.

## Project Structure

The project is organized as follows:

- **src/**: Contains the main application code.
  - **app/**: Application screens and layout.
  - **components/**: Reusable components for the application.
  - **constants/**: Static data used throughout the app.
  - **context/**: Context providers for state management.
  - **hooks/**: Custom hooks for managing logic and state.
  - **services/**: Functions for handling storage and sharing.
  - **types/**: TypeScript types for data structures.
  - **utils/**: Utility functions for various operations.
  
- **assets/**: Contains static assets like fonts.
- **__tests__/**: Unit tests for components and hooks.
- **app.json**: Configuration settings for the Expo application.
- **babel.config.js**: Babel configuration settings.
- **package.json**: Lists dependencies and scripts for the project.
- **tsconfig.json**: TypeScript configuration file.
- **.gitignore**: Specifies files to be ignored by version control.
- **.eslintrc.js**: ESLint configuration settings.
